# Driver Finder Mini App

## 1. How to Run the Project

Follow these steps to set up and run the application locally:

### **Clone the Repository**

```bash
git clone https://github.com/Savidya-Nirthana/Driver-Finder-Mini-App.git
```

### **Navigate to the Project Directory**

```bash
cd Driver-Finder-Mini-App
```

### **Install Dependencies**

```bash
npm install
```

### **Start the Application**

```bash
npm start
```

Use the **Expo Go** mobile application to scan the QR code and run the app on your device.

---

## 2. Tools Used

- **Visual Studio Code** – Code editor used for development.
- **Expo Go** – For running the React Native application on mobile.
- **Firebase Authentication** – Used for user login and authentication.
- **React Native** – Framework used to build the mobile application.
